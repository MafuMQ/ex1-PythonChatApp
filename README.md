# Project Overview for Python Chat Application

## Features
- Real-time messaging using WebSockets.
- User authentication and management.
- Supports multiple chat rooms.
- Allows file sharing and media embedding.
  
## Project Setup
1. Clone the repository:
   
   ```bash
   git clone https://github.com/MafuMQ/ex1-PythonChatApp.git
   cd ex1-PythonChatApp
   ```
   
2. Install the required packages:
   
   ```bash
   pip install -r requirements.txt
   ```
   
3. Set up the database:
   
   ```bash
   python setup_db.py
   ```

## Running Instructions
To run the application, execute the following command:

```bash
python app.py
```

The server will start and you can access the application at `http://localhost:5000`.

## Usage Example
1. Open your web browser and navigate to `http://localhost:5000`.
2. Create an account or log in if you already have one.
3. Join a chat room and start communicating with other users!
